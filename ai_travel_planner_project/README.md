# AI Based Travel Planner

This project demonstrates an AI-style travel planner that reuses an existing knowledge base for:
- Tourist places
- Food recommendations
- Personalized tour plans
- Cost assessment

## Files
- `knowledge_base.py`
- `destination_recommender.py`
- `cost_estimator.py`
- `personalized_planner.py`
- `main.py`
- `test_travel_planner.py`

## How to run
```bash
python main.py
```

## How to test
```bash
python -m unittest -v
```
