# Artificial Intelligence Laboratory Projects

This repository contains implementations and mini-projects related to core Artificial Intelligence concepts, including:

- Search algorithms
- Knowledge-based systems
- Knowledge graphs
- Bayesian networks

The projects demonstrate practical applications of AI techniques such as search, inferencing, probabilistic reasoning, recommendation systems, and knowledge representation.

## Projects Included

| Project | Description |
|---|---|
| Minimax and Search Algorithms | Implementation of Minimax, Alpha-Beta Pruning, Heuristic Alpha-Beta, and Monte-Carlo Tree Search |
| AI-Based Travel Planner | Personalized travel recommendation system using reusable knowledge bases |
| Knowledge Graphs | Introduction to Knowledge Graphs and tools used for KG development |
| Bayesian Networks | Probabilistic reasoning and inferencing using Bayesian Networks |

## 3. Knowledge Graphs

### Overview

A Knowledge Graph represents entities and relationships in a structured graph format.

### Example

```text
Harsh --studies--> Artificial Intelligence
Artificial Intelligence --belongs_to--> Computer Science
```

### Applications

- Search engines
- AI assistants
- Recommendation systems
- Semantic search

### Tools Explored

| Tool | Purpose |
|---|---|
| Neo4j | Graph database |
| Protégé | Ontology editor |
| Apache Jena | Semantic web framework |
| GraphDB | RDF graph database |
| RDFLib | Python RDF library |

## 4. Bayesian Networks

### Overview

Bayesian Networks are probabilistic graphical models used for reasoning under uncertainty.

### Features

- Probabilistic inferencing
- Graph-based modeling
- Dependency representation
- Decision support

### Tools Explored

| Tool | Purpose |
|---|---|
| pgmpy | Python Bayesian modeling |
| GeNIe | GUI Bayesian modeling tool |
| Netica | Probabilistic reasoning |
| BayesiaLab | Enterprise Bayesian analytics |

### Example Implemented

A Rain-Sprinkler-GrassWet Bayesian Network using `pgmpy`.
