
#  Knowledge Graphs (KG)

A **Knowledge Graph (KG)** is a structured way of representing information using:

- **Nodes** → Entities (Person, Place, Product, Concept)
- **Edges** → Relationships between entities

Knowledge Graphs help AI systems understand context and relationships instead of storing plain data.

---

#  Example

```text
Harsh ── studies ──> Artificial Intelligence
AI ── belongs_to ──> Computer Science
```

---

#  Applications

-  Search Engines
-  AI Assistants
-  Recommendation Systems
-  Healthcare
-  Business Intelligence

| GraphDB | Semantic graph database |
