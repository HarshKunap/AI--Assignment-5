
# 🧠 Bayesian Networks

A **Bayesian Network (BN)** is a probabilistic graphical model used to represent uncertain knowledge using:

- **Nodes** → Random variables
- **Edges** → Probabilistic dependencies

Bayesian Networks help in reasoning, prediction, diagnosis, and decision-making under uncertainty.

---

# 📌 Features

- Probabilistic reasoning
- Handles uncertainty
- Graph-based representation
- Supports inferencing and prediction

---

# 🚀 Applications

- 🏥 Medical Diagnosis
- 🤖 AI Systems
- 📈 Risk Analysis
- 🔐 Cybersecurity
- 🌦️ Weather Prediction

# 🧪 Example Implemented

This project implements a simple **Rain-Sprinkler-GrassWet** Bayesian Network.

### Scenario

- Rain affects whether the sprinkler is used.
- Rain and sprinkler together affect whether grass becomes wet.
